=== Dashboard ===

Dashboard - веб-сервис для статического и динамического отображения информации из базы данных.
Программный пакет сервиса содержит готовый демонстрационный набор виджетов и симулятор активности в базе данных.

Для установки сервиса на ПК самостоятельно используется инструкция (файл install_instruction.txt). 
Для установки специалистом со стороны разработчика сервиса необходимо обеспечение удаленного доступа к ПК (RDP).

ПО, необходимое для установки сервиса:
1. Git
2. Python 3.10.8 (along installation set enable "Add Path")
3. База данных MS-SQL/Postgres

На стороне клиента (пользователя сервиса) необходим веб-браузер. 
Работа сервиса протестирована на веб-браузерах Google Chrome, Mozilla Firrefox.
